# AI Resume Analyzer

An AI-powered web application that analyzes resumes and compares them with job descriptions using Natural Language Processing and Machine Learning.

The system extracts skills from resumes, evaluates ATS compatibility, predicts job roles, and suggests improvements to make resumes stronger.

---

## Features

- Resume PDF parsing
- Skill extraction using NLP
- Skill visualization
- ATS match score calculation
- Missing skill detection
- Resume improvement suggestions
- Job role prediction using ML
- Resume strength score with rating
- Interactive Streamlit UI

## Installation

Clone the repository:
git clone https://github.com/PrateekSingh-01/ai-resume-analyzer.git


Navigate to the project folder:
cd ai-resume-analyzer


Install dependencies:
pip install -r requirements.txt


Run the application:
streamlit run app.py


---

## Example Usage

1. Upload your resume in PDF format
2. Paste a job description
3. The system will:
   - Detect skills
   - Show ATS match score
   - Predict job role
   - Identify missing skills
   - Suggest improvements
   - Generate a resume strength score
